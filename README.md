# Team Snack Menu

Welcome to our collaborative snack menu! This repository showcases our team's favorite snacks organized by category.

## How to Use
- Browse the `menu/` folder to explore different snack categories
- Each file contains a curated list of snacks with descriptions
- Feel free to add your own snack category by creating a new Markdown file in the `menu/` folder

## Project Structure
```
pm-snack-menu/
├── README.md
└── menu/
    ├── _overview.md
    └── icecream.md
    └── candy.md
    └── mocktail.md
```

Happy snacking! 🍿
